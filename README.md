# 🍅 每日督学小助手

一个可爱的每日学习打卡应用，支持学习计划导入、智能进度结转、番茄钟倒计时、学习复盘统计等功能。

## ✨ 功能亮点

- 🎯 **今日任务首页**：启动即进入今日任务页，只显示当天的学习任务
- 📄 **学习计划导入**：支持导入 `.txt` / `.docx` 格式的计划文档，按 Day 分块解析
- ✏️ **手动计划录入**：也支持手动逐日录入学习计划
- 🧠 **智能进度结转**：根据以往「计划量 - 实际量」自动增减当天计划
- 🍅 **番茄钟倒计时**：支持暂停、结束，结束后一键记入复盘
- 📊 **学习复盘**：今日时间占比饼图 + 整体进度柱状图
- 💖 **可爱界面**：奶油粉色系，随机鼓励语，实时问候语
- 📅 **自动跨天**：每30秒与电脑时间对时，跨天自动切换到新的一天

## 🚀 快速开始

### 方法一：直接运行（推荐）

```bash
# 1. 克隆项目
git clone https://github.com/yourusername/study-tracker.git
cd study-tracker

# 2. 安装依赖
pip install -r requirements.txt

# 3. 运行应用
python app.py
```

### 方法二：双击运行（Windows）

1. 下载项目压缩包并解压
2. 安装 Python 3.8+
3. 双击运行 `run.bat`

## 📋 使用说明

### 1. 创建学习项目

在「📚 学习计划」页，输入项目名称（如「日语」「舞蹈」），点击「添加项目」。

### 2. 导入学习计划文档

支持的文档格式：

**格式一：Day 分块（推荐）**
```
Day1
单词记忆,30
语法练习,20

Day2
听力训练,25
阅读理解,30
```

**格式二：列表分点**
```
[ ] Day 1：单词记忆｜语法练习｜听力训练
[ ] Day 2：阅读理解｜口语练习
```

**格式三：日期排课**
```
7.25 瑜伽 16:00-17:30
7.26 雅思听力 19:00-20:00
```

### 3. 每日打卡

- 首页自动显示当天的学习任务
- 点击「▶」开始计时，点击「⏸」暂停，点击「⏹」结束并记录
- 完成任务后点击「打卡」标记完成

### 4. 学习复盘

- 在「📊 学习复盘」页添加复盘记录
- 查看今日时间分布和整体学习进度图表
- 双击记录可直接修改

## 📁 项目结构

```
study-tracker/
├── app.py                    # 主程序
├── requirements.txt          # 依赖清单
├── .gitignore               # Git忽略规则
└── example/                 # 示例数据（首次运行自动创建）
    ├── daily/               # 每日打卡数据
    ├── projects.json        # 项目配置
    ├── settings.json        # 系统设置
    └── [项目名]/            # 各项目的计划文档
        └── plan_*.txt/docx  # 导入的计划文件
```

## 🛠️ 技术栈

- Python 3.8+
- Tkinter（GUI界面）
- matplotlib（图表统计）
- Pillow（图片处理）
- python-docx（Word文档解析）

## 📝 注意事项

- 数据默认存储在程序所在目录的 `example/` 文件夹中
- 可在「⚙️ 系统设置」中修改数据存储路径
- 学习计划文档支持 `.txt` 和 `.docx` 格式
- 修改数据文件夹后需要重新启动应用生效
